# ☕ Coffee Heaven – Coffee Shop Website

A modern, responsive Coffee Shop website built with React and Vite.  
Coffee Heaven delivers a smooth and elegant user experience for showcasing coffee products, desserts, events, and reservations in a visually appealing layout.

---

## ✨ Overview

Coffee Heaven is a frontend-focused web application designed for a coffee shop brand.  
It provides a clean UI, organized sections, and a fully responsive layout suitable for all devices.

The project focuses on modern UI/UX design, reusable components, and smooth navigation.

---

## 🚀 Features

- ☕ Hero section with modern landing design
- 📖 About section introducing the brand
- 🍽️ Menu section (coffee + food items)
- 🍰 Desserts showcase (cheesecake, chocolate cake, tiramisu)
- 📸 Gallery section with images
- 🎉 Events section (birthday, family, special moments)
- 📅 Reservation section UI
- 🧭 Fully responsive navigation bar
- 📱 Mobile-first responsive design
- 🎨 Clean and modern UI design
- 🧩 Reusable React components architecture
